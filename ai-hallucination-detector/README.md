# AI Hallucination Detector

An AI-powered web app that detects hallucinated or fabricated responses generated by AI models in real time.

## Features
- Paste any AI-generated text and get an instant hallucination verdict
- Shows Hallucination vs Factual score with confidence percentage
- Flags suspicious signals like vague citations, overconfident language, unverified statistics
- Clean dark UI built for presentation

## Tech Stack
- Python + Flask (Backend)
- Transformers / Zero-Shot NLP Model (AI Detection)
- HTML + CSS + JavaScript (Frontend)

## Setup & Run

### 1. Install dependencies
```
pip install -r requirements.txt
```

### 2. Run the app
```
python app.py
```

### 3. Open in browser
```
http://localhost:5001
```

## How it works
1. User pastes an AI-generated response
2. The text is sent to the Flask backend
3. A zero-shot NLP classification model checks if it sounds fabricated
4. Heuristic checks flag vague citations, overconfident phrases, unverified statistics
5. Result is shown with confidence score and detailed flags
